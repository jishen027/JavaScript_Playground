## Http

### what is Http
- communication between web server & client
- Http request / response


Http is stateless
- every request is completly independent

Https
- Data send is encrypted
- SSL/TLS
- install certificate on web host

 Http Methods
 - GET
 - POST
 - PUT
 - DELETE

Http header fields
- General
- Response
- Request

HTTP status codes
- 1xx: information
- 2xx: success
- 3xx: redirect
- 4xx: client error
- 5xx: sercer error

- 200 - ok
- 201 - ok created
- 301 - Moved to new URL
- 304 - mot modified(cached version)
- 400 - bad request
- 401 - unauthorized
- 404 - not found
- 500 - internal server error
